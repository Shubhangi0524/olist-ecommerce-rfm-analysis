Olist E-commerce Data Analysis (RFM Segmentation)

🔍 Project Overview
This project analyzes customer behavior using the Olist e-commerce dataset.
RFM (Recency, Frequency, Monetary) analysis is used to segment customers and identify high-value groups.

🎯 Objectives
- To identify high-value customers.
- To analyze sales trends.
- To understand customer purchasing behavior.
- To provide business recommendations.

🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook

📈 Key Insights
- Loyal customers generate the highest revenue.
- The majority of customers fall into the Average segment.
- A low number of Champions indicates a growth opportunity.
- Seasonal trends observed in sales.

📊 Dashboard Features
- KPI metrics (Revenue, Orders, Avg Order Value)
- Customer segmentation (RFM)
- Monthly sales trends
- Product category analysis
